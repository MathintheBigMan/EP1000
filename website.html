Basic CSS

Having CSS to the HTML website gives the me the option to better stylize my website into a nice looking "diary" that helps me express the vibe of the content shown.

CSS as a programming language is described as a style sheet language that is used for describing the presentation of a document written with HTML.

Thing I learnt to do with CSS:

Basic Github

Github is a software distibution/hosting service for the users to upload their programs to and share it to other guthub users.
For my perpuse it is to host this website in order to make it accesible for others to browse throught.
These are the steps I took to set up my website in Github
  • 1. Making an account
  • 2. Search up a guide on "How to set up website on Github"
  • 3. Create a repository for the html code I want to publish
  • 4. Go to settings of the repository to select the repository to host
  • 5. Get link to share the wesite at Github Pages

After setting up the website I downloaded the Github desktop app in order to push updates to my website much easier to the repository. The two commands used to do this is the Connect to main and Fetch origin.
