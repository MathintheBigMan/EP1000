For this Assignment we are to create a usable phone stand that is creative and unique.

For my phone stand I wanted to prioratise the sound of the phone when it is on the phone stand itself. To achieve a better sound profile I made the inside hollow to let the sound bounce around much better.

Accesory Box

For this assignment I was tasked to make an accesory box with a shelf or a door/lid with hinges. The box should also have engraving on it

This assignment was pretty stright foward as I already had an idea in mind of a box with two layers, one with a shelf and the top being enclosed with as lid.

Final Parameters of the first Box: Final 3D model of the first box:

Process

For my accesory box I first started off with a simple cubed box shape as it was easier to create a dimention off. I then filled it in with the features I wante such as the "shelf" and top lid.

Initial Parameters of the first Box: Intial 3D model of the first box:
Making the "joints of the box". I first start a sketch from a flat layer. I then ofset form the edge by the thickness I want. I then make a 2D box from the edge and keyed in the value of the width using the parameter. Lastly I used the feature "Rectangular Pattern" to duplicate the parameters. I then cut out the boxes in between. I did the same technique for the rest of the box panels.

However, to save time on the other interlocking panels, I used a feature called "Combine". This allows me to cut any intersecting pieces so that one would fit through the other. I made sure to tick the Keep Tools option as it is disabled by default and would remove the "cutting" piece other wise. Making of the shelf component. This part took a while to make as it is essentially a smaller version of the big box that should be able to slide in and out of the container without risk of dropping out.

For this I used a simple method that extends the fingers out to use then as rails to guide the sliding box.
