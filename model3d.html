Mug

This model was created using a feature called revolve, this feature helps us make a circular shape using 2D drawings. This is done by creating a flat section of the circular object and revolving from there. Sketched used for the mug

Hex Nut

For this model we used the revolve funtion as we did for the mug however to make the inside compatable with screws we had to use the "Hole" funtion in order to make the shape automatically.

The tutorial for this piece is [here].
